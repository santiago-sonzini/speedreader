# Delta Speed Reading

**Delta Speed Reading** is a minimalist speed reading tool designed for efficiency and focus. Paste any text, set your reading speed in words per minute, and start playback to read one word at a time, centered on the screen with its middle letter highlighted for visual stability. The interface includes real-time progress tracking by time and word count, clear play and pause controls, and an automatic calculation of total reading duration, allowing you to maintain a consistent rhythm and measure performance with precision.
